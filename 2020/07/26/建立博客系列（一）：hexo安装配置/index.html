git

你要熟悉git并在你的电脑里安装git，git的详细知识可以参考廖雪峰老师的git教程

其实这里也用不到太多的git命令，不用过于担心

搭建Hexo

hexo是一款基于Node.js的静态博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装node.js

简单的讲node是一个Javascript运行环境，下载地址node.js,选择Windows.msi,64位或32位即可，安装的时候除了更改安装位置一路Next即可。

打开git bash,输入 node -v,npm -v,出现下图即代表安装成功。

npm

npm是随同Node.js一起安装的包管理工具,我们可以使用它从服务器上下载其他的程序，例如这里hexo即可使用Npm下载，但由于安装国外的东西速度会很慢，所以这里安装淘宝NPM，代替npm使用。安装后使用cnpm代替原来的npm命令即可。
