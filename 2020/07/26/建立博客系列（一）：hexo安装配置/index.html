建立博客系列（一）：hexo安装配置

发表于 2020-07-26 | 分类于 建立博客 | 阅读次数:

此教程采用hexo+coding搭建博客

git

你要熟悉git并在你的电脑里安装git，git的详细知识可以参考廖雪峰老师的git教程

coding

coding和github类似，因为coding是国内的，访问速度比github快

注册

1、进入coding 点击注册

2、填写好团队名称，团队域名，点击下一步

3、用自己的微信扫描二维码，接下来就完成了注册

创建项目

1、点击创建项目

2、这里选择DevOps项目，其它的项目模板会隐藏静态网站的构建，对后面的部署不方便，所以这里选择DevOps项目

3、填写项目名称，项目标识会自动生成，然后点击完成创建

添加SSH公钥

我们克隆项目有两种方式https和SSH方式，https方式每次操作都需要输入Coding的账号和密码，不方便，所以我们采用SSH方式

1、打开git bash，输入如下命令创建SSH key

ssh-keygen -t rsa -C “your_email@youremail.com“

双引号里的邮箱改成你自己的邮箱，然后一路回车，会出现一张萌萌哒图

2、如果一切顺利的话，可以在用户主目录（C:\Users\用户名）里找到.ssh目录，里面有id_rsa和id_rsa.pub两个文件，这两个就是SSH Key的秘钥对，id_rsa是私钥，不能泄露出去，id_rsa.pub是公钥，可以放心地告诉任何人。

3、打开Coding，点击个人账户设置

4、点击SSH公钥，然后点击新增公钥

5、在公钥内容上填写id_rsa.pub文件的内容

Hexo

hexo是一款基于Node.js的静态博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装node.js

简单的讲node是一个Javascript运行环境，下载地址node.js,选择Windows.msi,64位或32位即可，安装的时候除了更改安装位置一路Next即可。

打开git bash,输入 node -v,npm -v,出现下图即代表安装成功。
